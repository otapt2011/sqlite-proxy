🗄️ SQLite Proxy (sql.js + Redis)

REST API over SQLite with Redis persistence.

📡 Endpoints

GET /api/db?table=users

List all rows.

GET /api/db?table=users&id=1

Get one row by id.

POST /api/db?table=users

Insert – body {"record":{"name":"John","email":"john@x.com"}}

PUT /api/db?table=users&id=1

Update – body {"updates":{"name":"NewName"}}

DELETE /api/db?table=users&id=1

Delete row.

🔑 Authentication

Requires X-API-Key header.
